# Bricked's Nix config

A simple Nix config for me and my laptop <3

## Highlights

- GNOME-like Hyprland shell using [Mithril Shell](https://github.com/AndreasHGK/mithril-shell) and [Anyrun](https://github.com/anyrun-org/anyrun)
- Aggressive styling using [Stylix](https://github.com/nix-community/stylix)
- Home row mods using [Kanata](https://github.com/jtroo/kanata)
- Secure boot and TPM disk unlock using [Lanzaboote](https://github.com/nix-community/lanzaboote)
- Disk partitioning using [Disko](https://github.com/nix-community/disko)
- Binary cache using [Cachix](https://www.cachix.org)
