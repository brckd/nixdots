# AgsDots

## Getting Started

### Installing AGS and Bun

```bash
home-manager switch --flake .
```

### Installing type definitions

```bash
bun install
```

### Building

```bash
bun run build
```

## Acknowledgements

This project was created using `bun init` in bun v1.0.33. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
