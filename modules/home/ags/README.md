# AgsDots

## Getting Started

### Installing AGS and Bun

```bash
home-manager switch --flake .
```

### Installing type definitions

```bash
bun install
```

### Building

```bash
bun build
bun watch # rebuilds when files change
```

### Running

```bash
bun start
bun dev # restarts when files change
```
